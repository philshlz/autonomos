# showcases/ - proving the state survives the model

these are demonstrations, built for reach and clarity, not formal proof. each one is designed so the **owned-state** insight is the first thing a viewer encounters, never a support-bot-with-escalation-rules that buries the point.

the rule for every showcase: lead with state, not containment. the story is never "we stopped the agent from escaping." the story is always "we swapped the model and the company did not notice."

| showcase | the one-line hook |
|----------|-------------------|
| brain-transplant | swap the model live under a running agent; it keeps going with full continuity |
| amnesia-clip | a hard rule, violated the instant state is discarded; then held when state persists |
| speedrun | compress weeks into minutes; show drift across many swaps, then no drift with the ledger |
| cross-product-lift | same handbook and state running across different agent frameworks |
| inheritance-handoff | a new operator inherits a running agent cold and picks up from the ledger |

lead vehicle: OpenClaw. cross-family swaps via the openrouter adapter; local path via ollama.
