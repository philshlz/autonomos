# openrouter adapter

boots an autonomos agent through OpenRouter, the cross-family path. this is the adapter that makes the brain-transplant and speedrun showcases possible: swap the model string and the same company keeps running on a different family.

- set the model via OpenRouter model id (e.g. anthropic/claude-opus-4-8, openai/gpt-5-2, google/gemini-...)
- the handbook and ledger do not change when the model does
- every ledger entry records which model was live, so the swap seam is auditable
- egress allowlist already includes api.openrouter.ai
